# findAHome 寵物旅館網站
Find A Home 是一個寵物訂房網站，當有毛小孩需要暫時寄養，或是家裡有太多的寵物空間閒置時，可以提供給那些需要暫時寄養的毛家長們。

## Requirement
- Java version: openjdk 11.0.12+
- PSQL version: 14.4
- npm version: 6.14.15
- Vue version: 2.6.11
